# 🏢 Agent Company

An AI-staffed startup framework. Give it any idea and 15 specialist agents collaborate through a 5-phase pipeline to produce a complete startup operating package.

## Quick Start

Say **`startup`** in Copilot CLI to launch.

Say **`startup express`** for just vision + strategy (phases 1-2 only).

## The Team

| Phase | Agents | Output |
|-------|--------|--------|
| 1. Foundation | CEO | Vision, investor memo |
| 2. Strategy | CTO, CFO, VP Product | Architecture, financials, PRD |
| 3. Execution | VP Eng, Design, Marketing, Sales, Growth, Data | Sprint plan, design system, GTM, metrics |
| 4. Operations | People, CS, Legal, DevOps, Security | Hiring, support, compliance, infra |
| 5. Board Review | CEO | Executive summary, 90-day plan |

You review at each gate before the next phase runs.

## What You Get

A `.company/` folder with 25+ artifacts: vision docs, PRDs, architecture, financial models, GTM playbooks, hiring plans, security posture, and a prioritized 90-day action plan.
